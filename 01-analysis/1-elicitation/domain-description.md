# Описание домена и ограничения

## Описание домена

**Бизнес.** «Апекс» — уличный картинг-центр, работающий третий год. Центр организует **групповые заезды** на картах по трассе. Заезд длится 15–20 минут (включая инструктаж) и проходит по одной из **конфигураций трассы**: короткая (для новичков) и длинная (для опытных). Каждый заезд ведёт **маршал-инструктор**, который проводит брифинг и следит за безопасностью.

**Процесс.** Расписание формируется на неделю вперёд: задаются дата/время старта, конфигурация, назначенный маршал и число доступных картов. Всего в парке **14 картов**. Для новичковых заездов лимит — не более 8 человек. Клиент видит свободные **слоты** заездов и **записывается**, указывая, нужна ли ему **экипировка напрокат** (шлем, подшлемник) или он придет со своей. До заезда клиент может **отменить** бронь. Текущая ручная запись (Telegram, маркерная экипировка) приводит к путанице, простоям картов и двойным бронированиям.

**Цель клиентского приложения.** Дать клиентам самостоятельную онлайн-запись на заезды (просмотр слотов, бронирование, выбор экипировки, отмена), исключив ручную путаницу. Управление расписанием и работу маршалов обеспечивает **существующая инфраструктура**; приложение потребляет её данные через API.

**Ключевые объекты домена:**

- **Заезд / слот** — конкретный выезд группы: дата/время старта, конфигурация трассы, маршал, свободно картов. Создаётся в существующей инфраструктуре; клиенту доступен только для чтения.
- **Конфигурация трассы** — короткая (до 8 машин) или длинная (до 14 машин). Справочные данные (read-only).
- **Маршал** — ведет заезд; для клиента — справочные данные слота (read-only).
- **Карт** — инвентарь (всего 14).
- **Запись (бронь)** — резерв места (картов) клиентом на конкретный слот: количество человек, потребность в экипировке, статус.
- **Клиент** — записывается на заезды, отменяет запись.

> Роли Маршал и Владелец/Администратор обслуживаются **существующей инфраструктурой**
> и в этот скоуп не входят.

| Роль | Описание | Цели | В скоупе |
| :-- | :-- | :-- | :-- |
| **Клиент** | Посетитель картинг-центра. Пользуется с телефона. | Увидеть расписание, отфильтровать слоты, записать себя/друзей, выбрать экипировку, отменить запись, получать напоминания. | **Да** |

## Ограничения

**Бизнес-ограничения**

- Всего **14 картов**.
- Новичковая группа (короткая трасса) — **не более 8 картов**. Опытная — до 14.
- Лимит мест слота = `min(свободные карты, потолок конфигурации трассы)`.
- **Политика отмен:** ранняя (безопасная) отмена — не позднее чем за **1 час** до старта. Поздняя отмена (<1 ч) фиксируется без освобождения места в системе, штрафов в MVP нет.
- **Оплата офлайн** — приложение лишь фиксирует запись.
- **Период списка слотов по умолчанию — 7 дней (R-027).** Более длинный период доступен через фильтр. Если расписания нет — показывается **empty state** («Пока нет доступных заездов»).
- **Отмена заезда центром / по погоде (R-008).** Если заезд отменяется из-за погоды, бронь не удаляется, а получает статус **«Отменён центром»**; клиенту приходит **push-уведомление**. Повторная запись запрещена. Действие отмены выполняется вне скоупа приложения.

**Технические / платформенные**

- **Mobile-first**: приложение рассчитано на телефон.
- Взаимодействие с **существующим бэкендом через API**.
- **Граница серверной интеграции (R-004).** Существующий бэкенд — **black-box источник истины**: гарантия «**0 двойных броней**» на его стороне. Приложение корректно обрабатывает отказ, если картов уже нет.
- **Данные инфраструктуры (R-015).** Канонической схемой считается контракт API. Миграция исторических данных вне скоупа.

**Границы скоупа**

| В скоупе клиентского приложения | Вне скоупа (существующая инфраструктура / Phase 2) |
| :-- | :-- |
| Регистрация/вход клиента | Админ-панель владельца, интерфейс маршала |
| Просмотр списка слотов на 7 дней | Создание и редактирование расписания |
| Бронирование (до 5 картов за раз) | Онлайн-оплата |
| Выбор «своя экипировка / прокат» | Оценка маршала и рейтинги — **Phase 2 (R-001)** |
| Отмена записи с правилом 1 часа | Программа лояльности |
| Push-уведомления (напоминания, отмена погоды) | Автоматический учет прогноза погоды |

## Глоссарий

| Термин | Определение |
| :-- | :-- |
| **Заезд / слот** | Запланированный групповой выезд на трассу. |
| **Конфигурация трассы** | Маршрут (короткая для новичков, длинная для опытных). |
| **Маршал** | Инструктор, проводящий брифинг и контролирующий заезд. |
| **Карт** | Гоночный автомобиль центра (всего 14). |
| **Экипировка** | Шлем и подшлемник (выдаются напрокат или клиент приносит свои). |
| **Запись (бронь)** | Резерв картов на слот. |
| **Отмена центром** | Отмена заезда из-за погоды/форс-мажора в инфраструктуре. Бронь получает статус «Отменён центром», отправляется push. |
| **MVP** | Набор функций к старту высокого сезона (через 2 месяца). |
