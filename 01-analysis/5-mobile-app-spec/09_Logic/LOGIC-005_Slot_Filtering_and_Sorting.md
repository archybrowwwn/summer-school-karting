# Фильтрация и сортировка слотов

**ID:** LOGIC-005  
**Тип:** Логика  
**Домен:** 09. Логики  
**Приоритет:** High  
**Статус:** Черновик  

## Обзор

Логика фильтрации и сортировки доступных слотов на главной странице.

### Фильтры
- Дата и время
- Тип карта (standard / racing / junior)
- Уровень пилота (beginner / intermediate / pro)
- Доступность (только свободные слоты)

### Сортировка
- По времени начала
- По цене (возрастание / убывание)
- По популярности

### User Story
> Как клиент, я хочу быстро отфильтровать слоты по удобному времени и типу карта,
> чтобы найти подходящий заезд за несколько секунд.
