# Фильтрация заездов

**ID:** LOGIC-005  
**Тип:** Логика  
**Домен:** 09. Логики  
**Приоритет:** High  
**Статус:** Актуален  
**Источник:** FR-38, R-012, R-027, `listSlots`

---

## Параметры запроса `listSlots`

| Параметр | Описание | Логика |
|----------|----------|--------|
| `date_from`, `date_to` | Диапазон дат | Дефолт: сейчас → +7 дней (R-027) |
| `route_type` | `novice` / `experienced` | OR внутри группы при мультивыборе |
| `instructor_id` | Маршал-инструктор | OR внутри группы |
| `only_available` | Только со свободными местами | `free_seats > 0` |

**Между группами — AND.** Внутри группы мультивыбора — **OR**.

Сортировка по умолчанию: `start_at` по возрастанию.

---

## User Story

> Как клиент, я хочу отфильтровать заезды по дате, типу трассы и маршалу,
> чтобы быстро найти подходящий слот.
